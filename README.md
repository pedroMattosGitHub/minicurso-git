## OBJETIVO
Esse repositório foi criado para praticar o uso do Git e GitHub, realizando versionamento de arquivos.

## O QUE FOI FEITO 
- Criação de repositório
- Criação de arquivos
- Realização de 3 commits
- Atualização de arquivo

## O QUE É GIT 
É um sistema de controle de versão que permite acompanhar alterações de arquivos.

## COMO FAZER COMMIT 
1. Adicionar arquivos (Staging Area)
   Prepara os arquivos para o próximo snapshot: " git add . "
2. Criar Commit: " git commit -m 'mensagem' "

## COMO ENVIAR 
1. Conectar ao repositório: " git remote add origin https://github.com/pedroMattosGitHub/minicurso-git"
2. Enviar: "git push -u origin main"

## COMMITS 
- Commit 1 : criei "arquivo1.txt"
- Commit 2 : criei "arquivo2.txt"
- Commit 3 : modifiquei o "arquivo1.txt"
